VARIABLES STRINGS

Anteriormente hablamos de textos en javaScript, ahora es momento de hablar de números y las operaciones más comunes:

var edad = 32

OPERADORES DE ASIGNACIÓN

  • = → Asigna un valor a una variable
  • += → Operador de incremento
  • -= → Operador de decremento
  • *= → Operador de multiplicación
  • /= → Operador de división

edad += 1
var peso = 75
peso -= 2
Peso → 73

Por ejemplo, comer un sandwich aumenta 1 kilo de peso:

var sandwich = 1
peso += sandwich
Peso → 74

Otro ejemplo, sería que jugar futbol disminuye 3 kilos:

var jugarFutbol = 3
peso -= jugarFutbol
Peso → 74

La manera de almacenar decimales de javaScript no es tan precisa, debido a que destina cierta cantidad de bites en la memoria ram de la computadora para asignar un decimal por lo que no termina de ser tan preciso.

Para solucionar este problema debemos hacer los siguiente accedemos al modulo global de matematicas de javaScript Math

Si queremos 'x' cantidad de numeros decimales usamos total.toFixed() que viene siendo un string. Si queremos pasarlo de nuevo a numero decimal, lo hacemos con parseFloat()
var precioDeVino = 200.3
var total = Math.round(precioDeVino * 100 * 3) / 100
var totalStr = total.toFixed(3)
vat total2 = parseFloat(totalStr)
