COMPARACIONES EN JAVASCRIPT

OPERADORES DE COMPARACIÓN

  • == → Compara igualdad de valor
  • === → Compara igualdad de valor y tipo de dato
  • != → Evalúa la diferencia de valores
  • !== → Evalúa la diferencia del valor y tipo de dato
  • > → Operador mayor que
  • < → Operador menor que

Existen varias maneras de comparar variables u objetos de javascript. En primer lugar asignamos X a un valor numérico y Y a un string

var x = 4, y = '4'

x==y -> true lleva los 2 valores a un mismo tipo de dato por ende es igual y verdadero


x===y -> false aqui compara los tipos una letra y un número lo cual 2 tipos de variable diferente, por eso el retorno de false


Cuando realizamos operaciones es recomendable usar el triple igual === Esto permite que javascript no iguale las variables que son de distinto tipo. RECOMENTACIÓN usarlo siempre que se estén comparando variables.
