VARIABLES STRINGS

Los STRINGS son cadenas de texto. Es posible invocar funciones directamente sobre las variables, conocidos como métodos. En los STRINGS tenemos lo siguientes métodos:

var nombre = 'Octavio', var apellido = 'Cuellar'

toUpperCase() → nos permite cambiar las letras de un string en mayuscula.

var nombreEnMayuscula = nombre.toUpperCase()
nombreEnMayuscula = 'OCTAVIO'

toLowerCase() → nos permite cambiar las letras de un string en minuscula.

var nombreEnMayuscula = nombre.toLowerCase()
nombreEnMayuscula = 'octavio'

chartAr() → nos permite conocer que letra o caracter se encuentra en la posición que le indiquemos. Dentro del paréntesis introducimos un parámetro numérico que determinará que posición del string queremos conocer.

var primeraLetraDelNombre = nombre.chartAr(0)
primeraLetraDelNombre = 'O'

length() → es un atributo que nos permite conocer la cantidad de caracteres de los que está compuesto un string. El tipo de valor que obtenemos es numérico no de tipo string como sucedía con chartAr()

var cantidadDeLetrasDelNombre = nombre.length
cantidadDeLetrasDelNombre = 7

concatenación() → es posible unir strings alojados en diferentes variables en una misma variable:

var nombreCompleto = nombre + ' ' + apellido

Hay otra forma más nueva de concatenar strings, tiende a ser un poco más clara. Se llama INTERPOLACIÓN DE TEXTO. se usa el signo peso $, los corchetes {} (dentro de estas llaves podemos escribir código javascript) y un caracter especial, la comilla invertida ` ` estas comillas invertidas nos permite Interpolar Variables. Funciona de la siguiente manera:

var nombreCompleto = `${nombre} ${apellido.toUpperCase()}`

Una cosa más que podemos hacer con los STRINGS es acceder a un SUB-STRING a un string más pequeño. Lo hacemos con la función str

var str = nombre.substr(1, 2)
str = 'ct'
