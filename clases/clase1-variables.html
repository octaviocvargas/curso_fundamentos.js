VARIABLES

Una variable es un espacio en memoria donde se almacena un dato, que pueden ser consultados y/o modificados durante el ciclo de vida del programa.

Para imprimir algo en la consola lo hacemos con console.log('Hola Mundo JS!') es la manera en que interactuamos con el usuario, darle información y datos que hayamos calculado en nuestro programa.

Para declarar una varible simplemente escribimos var nombre ← esta variable existe pero su valor no está definido.

Para interactuar con la variable hacemos lo siguiente:

var nombre
nombre='Octavio'

-- --

var nombre = 'Octavio', apellido = 'Cuellar'
var edad = 32

console.log('Hola, soy ' + nombre + ' ' + apellido)
console.log('Tengo ' + edad + ' años')
