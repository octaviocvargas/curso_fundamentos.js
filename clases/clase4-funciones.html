FUNCIONES

Las palabras reservadas del lenguaje no podemos usarlas en variables

Son pedazos de código reutilizables

var nombre = 'Octavio', edad = 32

function imprimirEdad(n, e) {
console.log(`${n} tiene ${e} años`)
}

imprimirEdad(nombre, edad)
imprimirEdad('Vicky', 28)
imprimirEdad('Erik', 24)
imprimirEdad('Dario', 27)

para esta variable vamos a escribir una FUNCIÓN para decir que cierto nombre tiene tantos años. Para definir una función usamos FUNCTION
